## 如何生成 markdown 目录

- 安装 vs 插件，Markdown All in One
- 安装好插件，进入文档对应目录，打开 markdown 文档，光标停在你想放目录的位置
- Cmd + Shift + p，打开命令输入框，输入 Create Table of Contents 回车，目录生成
